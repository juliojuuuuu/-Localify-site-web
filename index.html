Bienvenue sur Localify : ton lecteur musical auto-hébergé.

Démarrage rapide Fonctionnalités PM2 & Tailscale

🚀 Démarrage rapide

  1. Installer Node.js LTS (18+ recommandé, 20+ conseillé).
  2. Vérifier l'installation : node -v et npm -v.
  3. Installer les dépendances : npm install.
  4. Lancer Localify : npm start.
  5. Ouvrir : http://localhost:3000.

Compte par défaut : admin / admin123456 (à changer).

🎧 Ce que Localify propose

📁 Structure musicale recommandée

music/
└── Artiste/
    └── Album/
        ├── 01 - Titre.mp3
        ├── 02 - Titre.flac
        └── cover.jpg
  • 1 dossier par artiste, 1 dossier par album.
  • Numéroter les pistes pour garder l'ordre.
  • Ajouter cover.jpg ou folder.jpg dans chaque album.

⚙️ Exploitation stable & accès distant

PM2 (24/7 recommandé)

npm install -g pm2
pm2 start server.js --name localify
pm2 logs localify
pm2 restart localify

Tailscale (accès privé)

Utilise Tailscale pour accéder à Localify hors de chez toi sans exposer de port public.

Exemple : http://100.x.y.z:3000

🧯 Dépannage express

  • EADDRINUSE : le port 3000 est déjà utilisé.
  • Bibliothèque vide : vérifier music/ et sa structure.
  • Covers absentes : ajouter cover.jpg ou folder.jpg.
  • Widget bloqué : autoriser les popups navigateur.
